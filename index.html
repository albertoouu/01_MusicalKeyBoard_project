Musical Keyboard

Para tocar el piano se utiliza el teclado, si se quieren tocar las teclas blancas, o de un tono de separacion, se utiliza el renglon que contiene las teclas: z, x, c, v, b, n, m. Por otro lado si se quiere tocar las teclas negras, se utiliza el renglon que contiene las teclas: s, d, g, h, j. Se utilizo esta configuración por ser similar a la de un teclado real, como se muestra en la imágen. El teclado tiene un rango de una octava y funciona tambien al dar click a las teclas.
